# 🚗 DriveData – QR-Based Vehicle Information Management System

## 🚀 Overview
DriveData is a full-stack vehicle information management platform that allows users to securely register vehicles, store important details, and generate unique QR codes for instant verification.

The system enables quick access to vehicle data such as insurance validity, PUC dates, and owner information through real-time QR scanning.

---

## ✨ Features
- Secure user authentication using JWT
- Vehicle registration and management
- Unique QR code generation for each vehicle
- Real-time QR scanning using browser camera APIs
- Cloud-based media storage using Cloudinary
- Structured database for insurance, PUC, and ownership tracking
